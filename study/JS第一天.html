3.JavaScript是什么？由什么组成？

  • 是一种运行在客户端（浏览器）的编程语言

  • ECMAScript 规定JS的基础语法规范

  • Web APIs：

    • DOM 文档对象模型，用来操作界面上的HTML文档。如对网页元素移动，设置大小，添加删除等操作。
    • BOM 浏览器对象模型，用来操作浏览器。如网页弹窗、检测窗口宽度、存储数据到浏览器等操作。

5.变量

  • 变量是计算机中用于存储数据的容器或者是一个装东西的盒子

  • 变量是存储数据的容器，所以说声明一个变量接收函数的返回值

  • 要想使用变量，首先就要创建变量也称为声明变量或者定义变量

    • 声明变量由两部分构成，声明关键字和变量名 即 let age
    • 先声明,再赋值 let age age=18
    • 声明并直接赋值 let age = 18
    • 同时声明多个变量

  • 多个变量声明中间用逗号隔开，但是为了代码可读性通常一行只声明一个变量

    let name = '小帅', age = 18
    let name = '小帅'
let age = 18

  • JS语法规范 等号两边要空格

    let name = '小帅'

  • 可以使用模板字符串，嵌入JS表达式输出变量的值

    document.write(`她的名字叫${name}`)

6.变量扩展-数组

数组 Array：可以实现单个变量名声明一组数据，同时可以存放任意类型的数据。

声明语法：let names = ['小帅','小美','小哥哥','小姐姐']

数组是按顺序保存数据的，因此每个数据都有自己的编号/索引/下标。

获取数据时通过数组下标访问数据。

console.log(names[0]) // 小帅

元素: 数组中保存的每个数据都叫做数据元素

下标: 数组中数据的编号

长度: 数组中数据的个数,通过数组的length属性获得

console.log(PI)

8.数据类型

JS属于弱类型编程语言,变量属于什么数据类型,只有赋值了以后才知道是什么类型

JS中数据类型分为基本数据类型和引用数据类型两大类

基本数据类型:

numner数字类型、string字符串类型、boolean布尔类型、undefined未定义类型、null空类型

引用数据类型: object对象类型

在JavaScript中，双等号（==）和三等号（===）都是用于比较操作的运算符，但它们的行为有所不同。

双等号称为相等运算符, 比较时会进行类型转换

console.log(100 == "100"); // true
console.log(1 == true); // true
console.log(null == undefined); // true

三等号称为严格运算符，它在比较时不进行类型转换。

console.log(100 === "100"); // false
console.log(1 === true); // false
console.log(null === undefined); // false

模板字符串：用于拼接字符串和变量

使用模板字符串,可以${}嵌入JS表达式输出变量的值

// 外面用反引号`` 里面使用美元符号花括号${变量名}
const name = '小帅'
document.write(`他的名字是${name}`)

表示肯定或否定在计算机中对应的数据类型叫做布尔类型, 他有两个固定的值 true 和 false, 表示肯定数据用 true (真), 表示否定的数据用 false (假)

未定义类型是比较特殊的类型,只有一个值undefined 只声明变量不赋值的情况下, 变量的默认值就是undefined 如果检测变量是undefined，说明值没有传过来

JavaScript中的null仅仅是一个表示"无", "空", 或"值未知"的特殊值

null和undefined区别

undefined表示未赋值

null表示赋值了，但是内容为空
